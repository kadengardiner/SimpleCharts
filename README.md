# Simple Charts

A simple, MyCharts patient portal helper dashboard that allows users access to instructions on how to view/manipulate their medical information including scheduling and vieiwing upcoming appointments, viewing test results, view medications, view billing, view/send messages, and more. Meant to be used on another tab with MyCharts portal opened on its own seperate tab.

---

## Dependencies

- [React 19](https://react.dev/)
- [Vite 8](https://vitejs.dev/)

---

## Installation

1. Clone the repo:
   ```
   git clone https://github.com/kadengardiner/SimpleCharts.git
   cd SimpleCharts
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the dev server:
   ```
   npm run dev
   ```

4. Open `http://localhost:5173` in your browser.

---

## Deployment

To deploy to GitHub Pages:
```
npm run deploy
```
NOTE: To run locally you need the .env file with firebase config keys, please message Kaden Gardiner @ kadengardinercs@gmail.com for the file contents, but we advise just using our deployed website.
